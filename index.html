Groove Scout

React Node.js Frontend Dev
Complete

A unified vinyl record search platform that aggregates listings from multiple marketplaces into a single destination — so collectors and casual listeners can instantly discover availability, condition, and lowest price.

Role: Front-end Developer · Oasis

Project Details

Built with CSS, JavaScript, Node, and React. Groove Scout solves the frustration of hunting across dozens of marketplaces, forums, and private collections to find a specific vinyl record.

01Aggregates listings from multiple sites and sellers into one place
02Shows condition info and lowest price for each record
03Search by vinyl record, artist, or album
04Account system with favorites and listing views

Technical Stack

React for the frontend component architecture, Node.js for backend API handling, and CSS for a clean, minimal dark-themed UI centered around the search experience.

The homepage centers around a single search bar with the tagline "Discover Your Next Vinyl" — keeping the experience focused and frictionless.

03

Emerald City Pet Shelter

UX Design Dashboard Figma
Complete

A desktop animal health dashboard for shelter volunteers to log daily observations, medical appointments, vaccinations, and behavioral notes — replacing paper forms and spreadsheets for 30+ volunteers.

Role: Software Designer · Personal

Project Details

Designed to digitize Emerald City Pet Shelter's animal tracking process, improving accuracy, accessibility, and coordination. The dashboard tracks name, species, breed, intake date, and check-up history per animal.

01Competitive analysis — reviewed ezyVet and Pet Manager platforms
02Workflow mapping — two core flows: Health Dashboard & Log a Check-up
03Sitemap — from homepage to animal-level pet dashboards
04Low → High fidelity wireframes in Figma

Design Decisions

Branding uses Deep Green, Sage Green, and Sky Blue to evoke nature and calm. The typography system uses 4 heading levels plus body text for clear information hierarchy.

Check-up tracking captures date, weight, appetite/eating status (High/Normal/Low), and the volunteering conducting the check-up — all logged through a clean modal overlay.
